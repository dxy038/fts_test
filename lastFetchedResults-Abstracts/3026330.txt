The purpose of this paper is to describe the ethics module in a senior seminar for petroleum engineering students. The objective of the ethics module is to improve oral and written communication skills by asking the student to discuss the ethical aspects of a moral dilemma. Each student must select a character from a complex situation that has been designed to present a range of moral dilemmas for a large number of characters. The student must select one ethical issue and discuss it relative to one or more ethical codes. Rubrics are used to evaluate student performance.
