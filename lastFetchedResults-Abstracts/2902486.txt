This work falls within the general framework of the Natural Language Processing (NLP), which aims to deal with linguistic data automatically in software. The objective of this work was to perform a morphological analyzer. The main and central part of the latter is the morphological segmentation of components that represent branches of a root. This system works by using a corpus on which the treatment is carried out and a dictionary that is used as an element of verification of result to obtain an effective analysis with a low error rate and accurate results. Using the results for educational applications as part of a learning environment supported by PC vic &#8220;TELA&#8221;.
