The objective of the paper is to introduce an architectural framework and the modular approach for our Mobile based Educational Web Content Miner. The architecture to obtain the desired result has been generated taking kids requirements into account and using the concept of Sieve coupons. Using modular approach, our focus in this paper is to implement the architecture generated in mobile based query interface for kids using java. The filtration of web content has been done using Sieve coupons and security controller. Also we have discussed various security measures for obtaining secured material for the kids suiting their age. The various design issues related to the kids and educational content have also been discussed.
