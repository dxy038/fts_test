The TR4 is a high-speed computer, constructed with modern techniques, and suitable for commercial as well as for technical and scientific applications. The central processing unit can communicate directly, without additional intervening elements, with the usual input and output devices. The basic design of software and hardware allows the user to benefit from effective programming techniques and parallel processing without extra programming burdens.
