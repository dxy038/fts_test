In this article Web mining is introduced as an instrument to evaluate technology commercialization potential. Firstly, we discuss Web mining approach can embrace the need for the objective evaluation from online perspective: uniform resource locator (URL) analysis can get indicative information about the stakeholdersÂ´ origin; link analysis via social network analysis (SNA) allows identification of stakeholders online and can divide them into communities according to their relationship. The method is also able to find authoritative Web sites; Web content analysis allows social concern and sketchy public opinion on certain evaluation indicators to be investigated. Secondly, a feasible framework for Web mining approach is proposed step by step with some available software. At last, the development direction of the approach and further wok in application is mentioned.
