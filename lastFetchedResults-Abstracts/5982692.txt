The case study of EIA in land use planning provides new accordance for the coming new round of land use planning revision, which means, it will be of great necessity to set up a both scientific and subjective assessment system. The traditional assessing methods are too much objective, while ANN model, as a nonlinear quantified method, is able to eliminate the influence of subjective factors successfully. In this article, a land use planning EIA system with 21assessment indexes is formed and then the BP net is fully used to get the EIA value during the last round of the land use planning. This offers the basis for the action of this round of land use planning. The result shows that, in the course of the implementation of last round land use planning in Chengde, the EIA value rises in a waving way.
