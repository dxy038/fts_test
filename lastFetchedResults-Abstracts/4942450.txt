The objective of this research is to implement GUI platform which can be applied on various mobile devices. Because of rapid change and diversification of mobile device, increasing usersÂ´ expectation to graphical user interface, the difficulty given to mobile application developer is greatly increasing. In this paper, we designed and implemented WYNEX, GUI platform ported onto the mobile devices. WYNEX is a middleware platform to provide convenience for mobile application developers by supporting them to develop user applications easier and more simple ways under the low memory and poor computing power devices.
