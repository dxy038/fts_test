A new image-search method is proposed for the distributed database environment. An agent is introduced in a search method that offers the best search technology in distributed database utilizing the directory database as an index of the search. A system model has been constructed based on the categorization of data and on the cost evaluation of the communication lines. The total serviceability is of prime interest to the system. The objective of this research is to develop the technologies for an image-based virtual museum in Hamamatsu city. The brief description of a new plan is also presented
