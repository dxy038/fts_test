Web services make information and software available programmatically via the Internet and can be used to build blocks for applications. A composite Web service generally is built through multiple existing service components. This paper puts forward a service composition framework based on mobile agent and active network, which encapsulates atomic service into active packet (called service agent, SA) using mobile agent, and composes Web service by the different SAs combining. The objective of our framework is to support service description, service publishing, service discovery, service selection and service composition
