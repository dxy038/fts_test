The experiment was performed using the Ginga-NCL Virtual STB version 0.12.4, a implementation of Ginga-NCL designed with the objective of offering in virtualized environment, characteristics similar to set-top boxes commercial. Tests were conducted on a personal computer running the Ginga-NCL Virtual STB, with the following hardware specifications: Intel Core i5 2.27GHz processor, 4GB RAM, 500GB HD with WiFi and Ubuntu 13.04 operating system. Regarding the mobile device and its operating system, the smartphone Nexus 4 with the Android 4.4 was chosen.
