The main objective is to identify and implement measures which would prevent the interruption of electricity supply to the consumer in the face of severe disturbances in the power system (e.g. transmission line faults, forced outages of generating units, etc.). This aim is achieved by assessing the impact of different contingencies on the power system and enhancing the security through preventive or corrective control actions. The security of the system is described by a Security Index which is a scalar value based on several attributes of system behaviour
